Потрібно завантажити з інтернету:
Vosk API (~15MB): vosk-win64-0.3.45.zip з GitHub https://github.com/alphacep/vosk-api/releases
Українська модель (~50MB): vosk-model-small-uk-v3-small.zip з alphacephei.com https://alphacephei.com/vosk/models/vosk-model-small-uk-v3-small.zip

Завантажити vosk-win64-0.3.45.zip, розпакувати в vosk/ у проекті.
Завантажити модель, розпакувати в vosk/model/.
Скопіювати libvosk.dll в папку з .exe (або додати до PATH).
